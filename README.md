# THEMELIPT - LIPT Protocol Theme

Landing page profissional para o LIPT Protocol - Protocolo DeFi deflacionário na Polygon Mainnet.

## Tecnologias

- **Next.js 16** - App Router
- **TypeScript** - Type-safe
- **Tailwind CSS 4** - Styling
- **shadcn/ui** - UI Components
- **Framer Motion** - Animações
- **Recharts** - Gráficos

## Features

- Tema escuro personalizado (Ember + Neon)
- Animações CSS customizadas (fogo, glow, partículas)
- Design responsivo (mobile-first)
- Componentes Web3 (conexão de carteira, calculadora de swap)
- Seções: Hero, Ecossistema, Tokenomics, FAQ, Footer

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Deploy

Este projeto está configurado para deploy na Vercel.
